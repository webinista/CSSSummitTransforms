2D Matrix Demo

  • 1,0,0,1,0,0 = Identity matrix
  • 1,0,0,1,200,200 = Translate 200 pixels along the X axis and 200 pixels along the Y axis. [ESC TO RESET]
  • 2,0,0,.7,0,0 = We can scale up by using a value greater than one or scale down by using a value between 0 and 1 [ESC TO RESET]
  • Also have a 3D Matrix demo available. You can see how that works.
