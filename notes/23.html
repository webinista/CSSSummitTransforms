Matrix Multiplication: Compound transforms

  • Can also do compound transforms using the matrix or matrix3d functions
  • Must multiply the transform matrices you wish to apply.
  • Here we are going to scale the object by 45 degrees, and scale it to 1.5 times its size
  • Scaling matrix [SHOW IN 2D DEMO] 1.5 0 0 1.5 0 0
  • Rotation matrix [SHOW IN 2D DEMO] .7071, .7071, -.7071 .7071 0 0 these values are the sine and cosine of 45 degrees
  • Multiply them [show in 2d Demo] 1.0606, 1.0606, -1.0606 1.0606 0 0
